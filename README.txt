Hello and welcome to Mostafa Vahidi's Newton Interpolation Program.

To Run the File Generator:
- Run: {java FileGenerator [number of datasets]} in your command line
	within \NewtonInterpol\out\production\NewtonInterpol> path
- For example: C:\workspaceSchool\NewtonInterpol\out\production\NewtonInterpol> java FileGenerator 10

To Run the Newton Interpol. Program to evaluate on the file generated:
- Run: {java NewtonInterpol [name of file generated: "dataset.test"] [value for x to find y for]
- For example: C:\workspaceSchool\NewtonInterpol\out\production\NewtonInterpol> java NewtonInterpol dataset.test 3
